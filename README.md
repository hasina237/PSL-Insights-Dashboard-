# PSL Cricket Analytics Dashboard

## Overview

An interactive cricket analytics dashboard built with Python and Streamlit to analyze Pakistan Super League (PSL) ball-by-ball data. The project provides data-driven insights into player performance, fantasy cricket metrics, and match situations through interactive visualizations.

## Features

* **Batter Analysis** – Runs, strike rate, boundaries, and performance trends
* **Bowler Analysis** – Wickets, economy rate, and bowling effectiveness
* **Fantasy Points Analysis** – Player rankings based on fantasy scoring metrics
* **Death Overs Analysis** – Performance evaluation during the final overs of an innings
* **Player vs Player Analysis** – Head-to-head comparisons between batters and bowlers

## Technologies Used

* Python
* Pandas
* NumPy
* Streamlit
* Plotly
* Jupyter Notebook

## Dataset

PSL ball-by-ball dataset containing match events, player statistics, runs, wickets, and fantasy performance metrics.
